# Korea — Fixup Studio / 픽스업 스튜디오

Negocio de marketing digital para PyMEs locales coreanas.

**Posicionamiento:** tercero neutral y honesto (진단), no una agencia de reviews falsas.
**Embudo:** Naver (que te encuentren) → Instagram (que te recuerden) → Kakao (que vuelvan).
**Roles:** esposa = cara coreana / titular legal (사업자) · Nacho = builder técnico · Claude = operación repetitiva.
**Target inicial:** estética / 피부과.

## Estructura
- `Clientes/` — una carpeta por cliente activo.
- `Auditorias/` — auditorías 진단 realizadas (input de screenshots → reporte coreano).
- `Contenido/` — posts de blog e Instagram redactados, por cliente.
- `Reportes/` — reportes mensuales de resultados.
- `Plantillas/` — assets base (auditoría, carpeta-cliente, calendario de contenido).
- `Prospectos/` — listas de research por rubro/zona.
